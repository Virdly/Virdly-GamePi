# Virdly-GamePi

Не большая Операционная Система специально созданная для связки Waveshare RP2040 PiZero и Waveshare GamePi13

Присутствуют базовые функции:
Настройки
Кастомизация
Файловый менеджер (Ещё в разработке)
Программы (Ещё в разработке)


Функции Файлового менеджера:
1.Создание и удаление папок
2.Чтение .txt и .jpeg файлов
3.Редактирование и сохранение
